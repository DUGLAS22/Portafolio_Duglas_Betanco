# Chat Servidor y Login

## Contexto del proyecto

1. **Chat Servidor**: Simula una ventana de chat donde los usuarios conectados pueden escribir y enviar mensajes. Además, existe una interfaz de servidor donde se visualizan los usuarios conectados y sus interacciones.
2. **Login de Usuarios**: Una aplicación de login donde los usuarios pueden iniciar sesión o registrarse, con validación de usuarios contra una base de datos. Esta aplicación no está relacionada directamente con el proyecto de chat

## Tecnologías usadas

### Bases de datos
-mysql

### IDE
-netbeans

### Lenguaje
-java
