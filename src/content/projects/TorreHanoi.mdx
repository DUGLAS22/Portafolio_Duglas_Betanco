# Simulación de las Torres de Hanoi

## Contexto del proyecto

1. **Torres de Hanoi**: Esta aplicación simula el clásico problema matemático de las Torres de Hanoi, en el que el objetivo es mover una serie de discos de diferentes tamaños de una torre inicial a una torre final, utilizando una torre intermedia y siguiendo ciertas reglas. Los movimientos se muestran paso a paso para visualizar el proceso completo de resolución.
2. **Interfaz de Usuario**: La aplicación permite al usuario seleccionar el número de discos con los que desea trabajar y observar cómo la solución se despliega automáticamente. También se incluye una opción de pausa y reanudación para analizar cada movimiento.
